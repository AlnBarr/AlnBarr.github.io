C+s starts a search forwards

C+r starts a search backwards

C+s again to move to next result RTN to exit search


Cut and Paste

Point is what emacs calls the cursor. Mark is an invisible end of the region. Use the mouse to select a region. On the command line use C-Sapce or C-@ then use C-n or C-p to move forward or back. The arrow keys also work.

C-k cut to the end of the line

M-k cut to the end of the sentence

C-w cuts the region

M-w saves the region

C-y paste (yank) the cut text

Indirect buffers

An indirect buffer allows you to work in a seperate window and allows you to have multiple views of an outline. Text changes are always reflected in the base buffer

C-x 4 c

Creats a twin of the current buffer in a new window.

Kill the indirect buffer by making it active and C-x k then C-x 1 to close the extra window
